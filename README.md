# 🌐 Responsive Landing Page Website

## 📌 Project Overview

This project is a **modern and responsive landing page** developed using **HTML5 and CSS3**. It is designed as part of an internship task to demonstrate fundamental front-end development skills including layout design, styling, and basic user interface creation.

The landing page presents a clean and attractive interface with a navigation bar, hero section, call-to-action button, and footer.

---

## 🎯 Objectives

- To understand the structure of a web page using HTML
- To apply CSS styling for layout and design
- To build a responsive and visually appealing landing page
- To practice real-world web development workflow using VS Code and GitHub

---

## 🚀 Features

- 🧭 **Navigation Bar**
  - Includes links: Home, Features, Contact
  - Simple and user-friendly layout

- 🎨 **Hero Section**
  - Eye-catching gradient background
  - Welcome message and description
  - Centered content alignment

- 🔘 **Call-to-Action Button**
  - "Get Started" button for user interaction

- 📱 **Responsive Design**
  - Adapts to different screen sizes
  - Clean and readable layout

- 📄 **Footer Section**
  - Displays copyright information
